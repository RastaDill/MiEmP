# MiEmP
Minimum Emacs set for Python

# Requirements #
* OS: GNU/Linux
* Emacs version: >=26.1

# Installtion #
1. Copy config
``` shell
cp ./.emacs ~/
```
2. Run Emacs.
After start will be check installed use-package and install if they not

3. Install dependent packages
``` emacs-lisp
M-x package-install-selected-packages
```

4. Enjoy
